## Description

This is a simple **Calculator application** built using **Python**.
It performs basic arithmetic operations such as **Addition, Subtraction, Multiplication, and Division** through a user-friendly command-line interface.

## Features

* Addition
* Subtraction
* Multiplication
* Division
* Handles invalid inputs
* Simple and beginner-friendly code

## Technologies Used

* Python

## How to Run

1. Make sure Python is installed on your system
2. Clone the repository or download the file
3. Open terminal / command prompt
4. Run the program using:

   ```
   python calculator.py
   ```
